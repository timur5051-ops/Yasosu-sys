Проект: Система организации курсов

1. Проведённые тесты:

1.1. Тестирование с некорректными данными
Создание курса с отрицательной ценой
Создание курса с отрицательной длительностью
Создание курса с пустым названием
Результат система выявляет ошибки логируются в app_debug.log

1.2. Тестирование уникальности данных
Попытка создать двух студентов с одинаковым email
Результат SQLite генерирует исключение UNIQUE constraint failed

1.3. Тестирование ссылочной целостности
Запись студента на несуществующий курс
Результат ошибка FOREIGN KEY constraint failed

1.4. Тестирование граничных условий
Работа с пустыми списками
Деление на ноль при расчёте средней цены
Результат добавлены проверки в код

2. Найденные и исправленные ошибки:

2.1. Ошибка в функции calculate_average
Было обращение к course[3] без проверки индекса
Стало добавлена проверка if len(course) > 3

2.2. Отсутствие обработки пустых данных
Было деление на ноль при пустом списке курсов
Стало добавлена проверка if not courses return 0

2.3. Нет логирования ошибок
Было ошибки не фиксировались
Стало создан модуль logger.py все ошибки записываются в лог

3. Внедрённые улучшения:

3.1. Система логирования
Создан модуль logger.py
Логи записываются в файл app_debug.log
Логи выводятся в консоль в реальном времени
Уровни логирования DEBUG INFO ERROR

3.2. Обработка исключений
Добавлены try-except блоки во все CRUD операции
Валидация входных данных
Генерация понятных сообщений об ошибках

3.3. Документирование ошибок
Создан отчёт об отладке
Лог ошибок прилагается к отчёту

Техническая документация
Система организации курсов и тренингов

1. Архитектура системы:

1.1. Компоненты системы
- Модуль course.py - управление курсами
- Модуль student.py - управление студентами
- Модуль algorithms.py - алгоритмы обработки данных
- Модуль app.py - веб-интерфейс

1.2. Структура данных
Класс Course
  атрибуты title duration price students
  методы add_student get_info show_students

Класс Student
  атрибуты name email courses
  методы enroll get_info show_courses

2. Установка и запуск:

2.1. Требования
- Python 3.8 или выше
- Git для контроля версий
- Доступ в интернет для установки зависимостей

2.2. Установка
1. Клонировать репозиторий
   git clone https://github.com/timur5051-ops/Yasosu-sys.git
2. Перейти в папку проекта
   cd my-project
3. Установить зависимости
   pip install flask
4. Запустить программу
   py main.py

2.3. Запуск веб-приложения
1. Запустить Flask приложение
   py app.py
2. Открыть в браузере
   http://127.0.0.1:5000

3. Использование системы:

3.1. Создание курса
python
course = Course("Название", 40, 15000)

3.2. Создание студента
python
student = Student("Имя", "email@mail.com")

3.3. Запись на курс
python
student.enroll(course)

4. Алгоритмы системы:

4.1. Сортировка курсов
Алгоритм пузырьковой сортировки
Сложность O(n²)
Применение сортировка курсов по цене

4.2. Поиск курса
Алгоритм бинарного поиска
Сложность O(log n)
Применение поиск курса по названию
